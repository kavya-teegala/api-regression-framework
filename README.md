# API Regression Automation Framework

This project is a Java-based API automation framework designed to execute regression test cases using JSON-driven test data.

## Tech Stack
- Java
- Maven
- TestNG
- REST Assured
- JSON
- Git

## Project Structure

src/main/java
- controller
- service
- repository
- utils

src/main/resources
- testdata.json

src/test/java
- TestCaseApiTest.java

## Features
- JSON driven test execution
- Modular layered architecture
- API validation using TestNG

## How to Run

Clone repository:

git clone https://github.com/TeegalaKavya/api-regression-framework.git

Run tests:

mvn test
